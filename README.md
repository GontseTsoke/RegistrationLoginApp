👤 Author

Gontse

📌 Project Description

This is a Java console application developed in NetBeans that allows users to:

Register with personal details
Validate username, password, and cell phone number
Log in using registered credentials
Receive feedback messages based on login status
⚙️ Features
✅ User Registration
Username must contain an underscore (_) and be no more than 5 characters
Password must:
Be at least 8 characters
Include a capital letter
Include a number
Include a special character
Cell phone number must:
Start with +27
Follow South African format
🔐 Login System
Users log in with username and password
System verifies credentials
Displays:
Success message with user’s name
Error message if login fails
🧪 Unit Testing

The project includes JUnit tests to verify:

Username validation
Password complexity
Cell phone format
Login functionality
🛠️ Technologies Used
Java
NetBeans IDE
JUnit
▶️ How to Run
Open the project in NetBeans
Run the Main.java file
Follow the prompts in the console
